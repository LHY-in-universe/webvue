================================================================================

第1步: 下载补丁文件
-------------------
从服务器下载这个文件到您的电脑:

使用 scp 命令下载:

或使用 FTP/SFTP 工具下载


第2步: 在本地电脑上克隆仓库
--------------------------
打开您电脑的终端（不是服务器！），执行:

cd ~
git clone https://github.com/LHY-in-universe/webvue.git
cd webvue
git checkout bench


第3步: 应用补丁
---------------
cd ~/webvue
git am ~/Downloads/code_patch_to_push.patch


第4步: 推送到GitHub
-------------------
git push origin bench -f

完成！代码已推送到GitHub ✅

================================================================================
                    或者：直接让我推送（最简单）
================================================================================

如果您有本地Git环境，可以直接:

1. 在本地电脑克隆: git clone https://github.com/LHY-in-universe/webvue.git
2. 添加服务器: git remote add server root@175.178.24.56:/home/webvue  
3. 拉取更新: git fetch server bench && git merge server/bench
4. 推送: git push origin bench -f
